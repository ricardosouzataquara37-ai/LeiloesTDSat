
Sistema de Leilões Online.

Sobre o Projeto
Este sistema foi desenvolvido como uma solução prática para gerenciar o fluxo completo de um leilão. 
A aplicação permite que usuários cadastrados visualizem itens disponíveis, acompanhem lances em tempo real
 e participem da disputa por produtos, garantindo que o sistema identifique corretamente o maior lance para a arrematação de cada item.

O projeto foca na implementação de lógica de persistência de dados e manipulação de objetos em Java, utilizando o 
banco de dados MySQL para garantir a integridade das ofertas.

Tecnologias Utilizadas
Linguagem: Java (JDK 8 ou superior)

Banco de Dados: MySQL

Conectividade: JDBC (Java Database Connectivity)

Controle de Versão: Git e GitHub

Arquitetura: Orientação a Objetos (POO)

Funcionalidades Principais
Cadastro de Itens: Inclusão de produtos com descrição e valor inicial.

Sistema de Lances: Registro de ofertas por diferentes usuários.

Consulta de Status: Visualização de itens disponíveis, leilões encerrados e histórico de lances.

Validação de Regras: Garantia de que novos lances sejam sempre superiores ao valor atual.

Persistência de Dados: Armazenamento seguro de todos os lances e vencedores no banco de dados.
